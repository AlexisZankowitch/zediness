## Installation 
```bash
npm install
npm link
```

## How to use zed
```bash
zed help
        zed [command] <options>
    
        request ............ send http request using rpn
        mqtt ............... mqtt
        [...]
        version ............ show package version
        help ............... show help menu for a command
```
